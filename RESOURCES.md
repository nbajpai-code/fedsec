# Federal Cybersecurity Resources

## Official Government Resources

*   **NIST Computer Security Resource Center (CSRC)**
    *   [Website](https://csrc.nist.gov/)
    *   Access to all FIPS and SP 800-series publications.
*   **FedRAMP Marketplace**
    *   [Website](https://marketplace.fedramp.gov/)
    *   List of authorized cloud service providers and assessors (3PAOs).
*   **CISA Cybersecurity Division**
    *   [Website](https://www.cisa.gov/cybersecurity)
    *   Directives, alerts, and resources for critical infrastructure.
*   **DoD CMMC**
    *   [Website](https://dodcio.defense.gov/CMMC/)
    *   Official updates and documentation on the CMMC model.

## GitHub Repositories

*   **OpenSCAP**
    *   [Repo: OpenSCAP/openscap](https://github.com/OpenSCAP/openscap)
    *   *Description*: NIST-certified SCAP 1.2 toolkit. Open-source tool for automated configuration/vulnerability scanning.
*   **ComplianceAsCode**
    *   [Repo: ComplianceAsCode/content](https://github.com/ComplianceAsCode/content)
    *   *Description*: Security automation content (SCAP, Ansible, Bash) for various platforms (RHEL, Fedora, etc.), mapping to STIGs and NIST standards.
*   **CivicActions Compliance Tools**
    *   [Repo: CivicActions/compliancetools](https://github.com/CivicActions/compliancetools)
    *   *Description*: Tools to help generate SSPs and compliance documentation for ATOs.
*   **GovReady**
    *   [Repo: GovReady/govready-q](https://github.com/GovReady/govready-q)
    *   *Description*: Open source compliance server for automating RMF and FedRAMP artifacts.

## Standard Bodies & Other Organizations

*   **Center for Internet Security (CIS)**
    *   [Website](https://www.cisecurity.org/)
    *   CIS Benchmarks are often used as a baseline for systems.
*   **Cloud Security Alliance (CSA)**
    *   [Website](https://cloudsecurityalliance.org/)
    *   CCM (Cloud Controls Matrix) often maps to FedRAMP/NIST.
